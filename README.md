## Installation

Installation

```

npm install

```

Run test with Jest

```

npm run test

```

Run example LSP

If you want run LSP mode develop, run the next line:

```

npm run dev:lsp

```

Compile app to js

If you want run mode production, first compile app, run the next line:

```

tsc

```
